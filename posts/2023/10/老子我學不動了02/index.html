老子我學不動了(02): Raycast 真的好香

2023-10-17
learning

不知道我目前所待的公司文化到底是否「安全」、「開放」，有時候還是會不太敢分享一些有的沒的。於是這個系列可能算是變成我一種出口來開始做個分享吧！或許原來可以這麼寫的定位是「工作與技術」，但這個系列或許沒有像定位在這樣，就分享自己看見的新東西酷玩意兒吧 💪

W3CTech 問卷

https://w3techs.com/technologies/history_overview/content_management/all

忘了是什麼時候看到這個酷東西… WP 目前在 CMS 根本霸主，擁有快超過 40% 這種市占率，另外他也有其他例如程式語言、Hosting 等等的市佔率調查，看了覺得真的好好玩喔～ 大家有空可以在這個網站裡面四處晃晃

Facebook 看到的 #selena 商管文系列

還蠻喜歡看這種假掰的分析文章、類似大局觀的文章呢！看到作為老闆的為難、以及目前年輕世代對於工作的態度~

  • 留住好員工的方法，是告訴對方：有天你會離開我

我覺得就像是我之前看到某個技術大大講到：「我找人進來，是為了要來取代我」，如果從這個角度來看，我覺得 Selena 這篇文章的內容就和這位技術大大的觀點不謀而合！

  • 為什麼餐飲業外場請不到人？給餐飲業主的建議

這篇真的很讚，其實世界不斷在變化，過去的方式或許在過去可以「成功」，也不可能保證一再適用「現在」！總之對於餐飲業主的建議要不就是提高人力成本、不然就是改善服務流程

另外也額外推廣閱讀這位技術大大的幾篇文章(其實為什麼看到這篇文章就是來自那個棒棒的 Star Rocket 科技創業週報 呢！)：

  • 權威是把以血作為代價上古妖刀，出鞘必見血：超愛這一篇，我想到我以前的工作經歷！
  • 為什麼我們提出改變，常常不被上層同意？
  • 工程師的管理職實驗

居然還有幫忙開公司的公司

在晃 WP 開發日常的時候，看到這個東西: 簡單開公司

居然幫忙人家開公司也是一種商業模式！這真是使我驚奇…另外，看到他們工程師在幹嘛的文章，我好喜歡這種能為公司內部解決問題的氣氛氛圍，要不是公司開在台北…不然我也很想投遞試試看…

inside 文章快速整理

陸續看 inside 文章，突然也是有感而發…但就也一起寫下來整理看看…

  • 40 歲的哀愁：拼不過新鮮的肝靈活的腦，工程師面臨中年危機

感嘆…真不知道我還能吃 WP 這行飯可以吃多久、寫 code 這行飯可以吃多久…

  • 沒什麼是原創的：把知識連起來就是創意

我覺得真的是這樣，要把腦袋裡各種知識融會貫通，形成連結、形塑自己的價值觀！

  • 降級論：到不同的領域再打一次比較容易

這一篇有點讀不太懂…不過和我之前有位學長分享說不要進去資訊業當資訊部門人員，而是要去非資訊公司去當資訊部門人員的說法有點異曲同工之妙…

Raycast

意外發現到這個的東西，mac 的 spotlight 那時候就讓我心神嚮往不已，如今居然有這種更猛的工具…mac 工作效率 up up up ☝️

  • Raycast 教學

ByteByteGo 開源 System 101 教材

只能說這個資源也是很讚，雖然看過了以後感覺大部分還是有點理解，除了比較弱的就是「rpc」之類的需要再加強以外，感覺大部分都略有所聞，但開源就是給讚 👍

對了，他也有提供電子報的服務，但有些進階概念需要付錢…哈我要訂閱的服務真的太多了，所以只好就先 pass 哈哈😂
