老子我學不動了(03): 最近升級到 ChatGPT Plus 方案

2023-11-04
learning

看 YT 被燒到，原來 ChatGPT 提供了「plugin」方案，所以就不在當 ChatGPT 的免費仔了…

建議不要使用 WP-Rocket

最近讀到文章建議不要使用WP-Rocket…文章有具體說明為什麼他不建議使用WP-Rocket的具體理由，另外他的解決方案是可以聘用優秀的開發人員或者選擇更好的主機方案…

我笑死ＸＤ

  • 文章來源：https://meowapps.com/do-not-use-wp-rocket/

發現 WordPress Serverless 方案

最近看到這個專案：https://github.com/mitchmac/ServerlessWP

被前同事影響，對 Serverless 真的很有興趣跟熱情，看到有這個可以玩Serverless 的 WordPress 有夠給他心動…

其他人整理的後端資源

逛網路的時候看到有個人整理後端的相關資源，個人覺得還不錯就給他收錄起來了…當成我文章的材料ＸＤ 上網址：https://calpa.me/tools/back-end-developer-free-tools-resources/

主要發現有幾個很酷的東西：

  • pipedream 這個服務，我想我大概可以取消使用github action 的schedule啦 😂
  • https://www.warp.dev/ : 用 Rust 寫的 Terminal… 感覺可以提高自己使用黑畫面的效率…不過我個人還沒有玩，感覺對我有點沒有太多吸引力

升級 ChatGPT Plus 方案

狠下心，購買了 Plus 方案，最近還在研究可以怎麼玩他的 plugin 提升效率哈哈！
