Python 起手式：淺談tuples

Oct 8, 2018 1 分鐘閱讀 鐵人賽PythonX30python

tuples 也是Python的資料型態之一，同屬於“列表”的一種。宣告方式如下

1>>> tuple =('a','b','c');
2>>> tuple
3('a', 'b', 'c')

tuple 同樣與list一樣可以放置不同的資料型態，唯一不同的是兩者的宣告方式一個是用小括弧;另一個則是使用中括弧。兩者最大不同之處就是tuple不能修改、新增，而list可以。

 1>>> list = ['a','b','c','d']
 2>>> list[0]='c'
 3>>> list
 4['c', 'b', 'c', 'd']
 5>>> tuple=('a',0,2,'c')
 6>>> tuple
 7('a', 0, 2, 'c')
 8>>> tuple[0]='b'
 9Traceback (most recent call last):
10  File "<stdin>", line 1, in <module>
11TypeError: 'tuple' object does not support item assignment
