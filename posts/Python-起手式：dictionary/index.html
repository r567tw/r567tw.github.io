Python 起手式：dictionary

Oct 9, 2018 1 分鐘閱讀 鐵人賽PythonX30python

寫到這裡，真的覺得可以寫滿30天持續不間斷的真的超級強大的，我覺得自己快撐不住了。不過為了能夠學會Python，我仍然願意忍受工作的辛苦想耍廢之後，還是要繼續寫下去…… 同時，原本是想自己慢慢寫慢慢自己看的，沒想到居然有這麼多人訂閱這篇文。真是讓我受寵若驚，這也是我必須要繼續寫下去的理由。

Python 中也有一個資料型態，叫做“Dictionary”，有點像是php的 associative arraysphp 是這樣表示的：

1$age = array("Peter"=>"35", "Ben"=>"37", "Joe"=>"43");

而在Python當中，dictionary是這樣表示的

1age = {'Peter': 35, 'Ben': 37, 'Joe': 43}
2age['Peter'] # 35

而dictionary 和 list一樣可以修改

1age['Ben']=38
2age # {'Peter': 35, 'Ben': 38, 'Joe': 43}
3

至於dictionary的刪除或清空則需要這樣

1age = {'Peter': 35, 'Ben': 37, 'Joe': 43}
2del age['Joe']
3age # {'Peter': 35, 'Ben': 37}
4age.clear()
5age # {}

比較特別的是，dictionary可以轉換成string

1age = {'Peter': 35, 'Ben': 37, 'Joe': 43}
2len(age) # 3
3str(age) # "{'Peter': 35, 'Ben': 37, 'Joe': 43}"
4type(str(age))  # <class 'str'>
5type(age)  # <class 'dict'>
6

其他操作

1age = {'Peter': 35, 'Ben': 37, 'Joe': 43}
2age.get('Ben') # 37
3age.items() # dict_items([('Peter', 35), ('Ben', 37), ('Joe', 43)])
4type(age.items()) # <class 'dict_items'>
5age.keys() # dict_keys(['Peter', 'Ben', 'Joe'])
6age.values() # dict_values([35, 37, 43])
7

同場加映

1age = {'Peter': 35, 'Ben': 37, 'Joe': 43}
2list(age.keys())  #可以只取dictionary 的key的list
3# ['Peter', 'Ben', 'Joe']
4list(age.values()) #可以只取dictionary 的value的list
5# [35, 37, 43]
