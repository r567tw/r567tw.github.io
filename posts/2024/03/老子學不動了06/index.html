老子我學不動了 (06): Laravel 11 又更強了

2024-03-16
learning

Laravel 11 發布

最近看到 Laravel 11 發布，最近這一兩年沒什麼在寫 Laravel 了，但覺得很驚奇！

發現很多很有趣的 dx point, 感覺就很 artisan 啊…另外，也公布了官方 WebSocket Package…我覺得真是好玩且有趣…

好想回去寫 Laravel 啊ＸＤ

Python 也可以在 Html 上執行

看到這個有趣的專案：https://pyscript.net/ 他透過將 Python 解釋器轉換成 WebAssembly 形式，從而讓 Python 代碼能夠在瀏覽器端運行

讓我不經想著，或許有一天 PHP 也可以在 Client 執行這樣ＸＤ

以下是個火力展示 Video: https://www.youtube.com/watch?v=z5x1mP3vDo4
